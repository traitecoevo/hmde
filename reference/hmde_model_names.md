# Returns names of available models.

Returns names of available models.

## Usage

``` r
hmde_model_names()
```

## Value

vector of character strings for model names.

## Examples

``` r
# basic usage of hmde_model_names
hmde_model_names()
#> [1] "constant_single_ind" "constant_multi_ind"  "canham_single_ind"  
#> [4] "canham_multi_ind"    "vb_single_ind"       "vb_multi_ind"       
#> [7] "affine_single_ind"  
```
